---
title: "hugo-tools"
description: "Open-source tools for Hugo authors, theme makers, and editors."
---

Hugo is a great static-site generator with a sharp edge for editors:
front-matter is YAML / TOML / JSON, content is Markdown with
shortcodes, data lives in CSV / JSON, themes pull params from any of
three different files. CMS layers usually solve one slice and lose
the rest.

**hugo-tools** is the home of projects that try to fix that — a
desktop editor, a migration tool, a plugin marketplace. All MIT
unless noted, all built around the actual filesystem your `hugo`
command already understands.
